# Book Store Management System 📚

A **Book Store Management System** designed to streamline the management of books, authors, customers, and sales. This project allows store owners to track inventory, manage orders, and maintain detailed records efficiently.

## Features ✅

- **Book Management:** Add, update, and delete books with details such as title, author, genre, price, and stock quantity.
- **Author Management:** Maintain a database of authors, including name, biography, and list of published books.
- **Customer Management:** Register and manage customer details, purchase history, and contact information.
- **Sales & Orders:** Track orders, generate invoices, and manage payment records.
- **Inventory Tracking:** Monitor stock levels and automatically update after sales.
- **Search & Filter:** Easily search books by title, author, genre, or ISBN.
- **Secure Access:** Role-based access for admin and staff users.

## Tech Stack 🛠️

- **Backend:** Java, Spring Boot
- **Frontend:** Thymeleaf / HTML / CSS / Bootstrap (or React if applicable)
- **Database:** MySQL
- **ORM:** JPA / Hibernate
- **Build Tool:** Maven
- **Security:** Spring Security (Optional: JWT authentication)
- **Version Control:** Git & GitHub
